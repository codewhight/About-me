Stage 1 Boss

【劫火之影】· 熾焰

熾焰曾是戈歐勒大陸高空最優雅的巡航者，直到它將靈魂獻祭給了太古火龍。

它不再需要尋找棲息地，因為它所到之處皆為焦土。熾焰的雙翼融合了煉獄峽谷的岩漿，揮動間散發出的並非微風，而是足以融化克羅姆平原堅實地表的。它守護著通往煉獄的核心要道，那雙如紅寶石般燃燒的雙眼，正冷冷地俯瞰著那些試圖奪回文明火種的恐龍勇士。

「當黑羽遮蔽烈日，隨之而來的並非陰涼，而是焚盡靈魂的業火。」
Attributes
boss屬性: hp:100, speed:4, score:30
Skills - Page 1

boss技能:【獄火鳳煌・紅蓮葬送曲】

熾焰振翅釋放火焰，在戰場上劃開現實與虛無的裂縫。喚出的「噬魂鳳煌」不再是單純的生物，而是由劫火構成的審判因子。這曲葬送曲會根據戰場的崩壞程度，切換四種不同的「毀滅軌道」，將繼承者們的退路徹底焚毀。

「繼承了虛偽神力的卑微生命啊……在我的紅蓮之翼下，感受這份積壓了數百年的焦灼怨恨吧！」

Skills - Page 2

boss技能:【獄火鳳煌・紅蓮葬送曲】 - 斷罪橫軸

描述： 「將空間斬斷的獄火絲線，於水平線之彼端同步齊發。這是不可跨越的紅蓮之牆，挑戰者只能在整齊的火光中迎接寂滅。」

視覺感： 火鳥成群呈水平直線排列，如潮汐般規律掃蕩戰場。

Skills - Page 3

boss技能:【獄火鳳煌・紅蓮葬送曲】 - 肅清縱軸

描述： 「自穹蒼頂端降下的秩序之刃，垂直貫穿大地。在天理的制裁下，任何試圖閃避的動作皆是徒勞，只能在間隙中渴求生存的奇蹟。」

視覺感： 火鳥由上方垂直墜落，形成密集的火羽箭雨。

Skills - Page 4

boss技能:【獄火鳳煌・紅蓮葬送曲】 - 混沌亂流

描述： 「打破理性的桎梏，火鳥於橫向次元中瘋狂起舞。無序、混亂、狂暴——那是來自虛無的嘲諷，在無法預測的火影中，絕望將先於死亡抵達。」

視覺感： 火鳥以隨機的高度、參差不齊的時間點，狂亂地橫向突襲。

Skills - Page 5

boss技能:【獄火鳳煌・紅蓮葬送曲】 - 災厄雨落

描述： 「凋零的流螢化作無盡的厄運，自蒼穹各處無差別散落。沒有預兆、沒有邏輯，這是命運對脆弱生命的最終試煉，大地將在火雨的洗禮下化為焦土。」

視覺感： 火鳥從上方隨機位置、以不穩定的頻率紛紛墜落，封死地面躲避空間
