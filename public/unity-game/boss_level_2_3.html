Stage 2 Boss

【冥雷之影】· 雷霆

當雷霆踏入這片被詛咒的森林，他那銳利的電光被暗影徹底染指。原本純淨的藍色閃電，如今混合了幽魂的怨念，轉化為詭譎的冥府綠雷。

這些雷電不再發出震耳欲聾的轟鳴，而是伴隨著陣陣如泣如訴的鬼嚎。暗影力量不僅強化了他的速度，更讓他的攻擊具備了穿透靈魂的特性。他盤踞在佈滿古老符文的石堆上，四周徘徊的幽魂成了他的能量電池，源源不絕地為其供能。在暗影的加持下，雷霆能瞬間化作一道綠色的虛影消失在林間，並在下一個瞬間，從虛空中降下撕裂靈魂的毀滅重擊。

「當綠色的電火在林間閃爍，那是死神正在校準他的雷標。你逃不掉，因為影隨雷行，雷入魂滅。」
Attributes
boss屬性 hp:240, speed:12, score:45
Skills - Page 1

boss技能一:【影嘯冥雷・絕命魂標陣】

雷霆將自身力量融合幽魂雷鳴形成為極度不穩定的「影蝕電漿」。當他發出如鬼囓般的裂嘯時，綠色的雷光將穿透虛實邊界，在戰場上劃下死神的標記。這不再是單純的能量釋放，而是以雷電為媒介的靈魂收割協議，一旦被綠雷掃過，生者的靈魂頻率將被強行同步至凋零狀態。

Skills - Page 2

boss技能一:【影嘯冥雷・絕命魂標陣】 - 冥雷・斷魂真理軸

描述： 將幽魂的怨念壓縮至絕對的水平線，形成不可跨越的『死神校準線』。這是冥府規律的展現，任何試圖在真理軸線上停留的意志，都將隨著閃電的閃爍而徹底煙滅」

視覺感： 雷霆全身迸發出刺眼的慘綠色電弧，周圍的幽魂化作綠色的光點匯入其胸口。隨即，一道筆直、穩定且充滿靜電干擾感的綠色像素雷電光束橫向貫穿畫面。

Skills - Page 3

boss技能一:【影嘯冥雷・絕命魂標陣】 - 影嘯・九幽凶雷變

描述： 打破雷電的傳導規律，讓冥雷如同森林中的虛影般變幻莫測。當綠色電火在無序中狂亂跳動，那是命運在進行最終的隨機判決。

視覺感： 隨著雷霆的雙翼劇烈顫動，多道綠色像素雷電光束在不同的高度、以參差不齊的時間點毫無規律地橫向突襲。

Skills - Page 4

boss技能二【影蝕雷災・噬魂雷雲陣】 - 雷霆發出震動地脈的鬼嚎，在戰場上空強行編織出一片「不祥雷雲」。這片雷雲混合了冥府的綠雷與森林的怨念，形成了一個無視物理法則、終日不散的詛咒力場。當雷雲成形，它將無視防禦，源源不絕地將壓縮至極致的「怨靈落雷」垂直降下。凡是被這股「天災雷雨」觸及的靈魂，生命將如焦土般乾枯、崩解，成為這場葬禮最微不足道的殘燭。

描述： 將太古的雷電與幽魂的怨念共鳴，形成絕對不可逆轉的『天譴之矛』。在此雷鳴聲中，世界將被強行同步至寂滅。

視覺感： 隨著雷霆的咆哮，戰場上空突兀地浮現出一片巨大的、慘綠色的不祥黑雲的落雷形式，但雷雲本身更具動態感。雷雲之中爆發出陣陣痛苦的幽魂哭喊聲。隨即，無數道筆直的、如極光般絢麗的慘綠色像素雷電光束，從雷雲中成群俯衝而下。

Skills - Page 5

boss技能三:【影羽驚雷 ・冥雷羽殛】 - 雷霆狂暴地煽動那對被暗影雷光纏繞的巨翼，將自身精血與森林怨氣凝結成「冥雷影羽」。這些羽毛在脫離翅膀的瞬間會釋放出高頻的綠色電漿，形成具備追尾與爆裂特性的死亡彈幕。

描述： 「將禁忌的雷電意志灌注於凋零之羽，在瞬息間編織出覆蓋因果的雷光之網。在這場由影與雷交織的洗禮中，汝之靈魂將如狂風中的殘燭，在萬千影羽的咆哮下徹底崩解。」

視覺感： 雷霆在半空中劇烈振翅，雙翼散發出幽暗的紫色光芒與刺眼的綠色電弧。隨即，無數枚帶電的「冥雷影羽」如暴雨般向前方射而出。羽毛周圍纏繞著扭曲的綠色像素電漿，並伴隨著數個尖叫的幽魂殘影。形成一片密不透風、閃爍著詭譎綠光的彈幕，將戰場化為一片無法逃脫的冥雷煉獄。
