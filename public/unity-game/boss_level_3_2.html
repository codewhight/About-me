Stage 3 Boss

【訊號殲滅者】· 雷霆

當雷霆踏入新星科技城的核心能源區，他並未摧毀那些複雜的電路，而是將自身原本狂暴的雷霆意志，與城中的超導機甲以及無限能源矩陣進行了深度的融合。

現在的雷霆，已然成為穿梭於數據與現實邊界的超維生命體。

他那融合了藍色離子能量的機甲羽翼，不僅能讓他在瞬間突破數倍音障，更使其每一次振翅都帶有精密計算後的電磁脈衝。他能隨心所欲地操控科技城的防禦系統，將無數高壓離子砲與數據流化作殲滅入侵者的無情利刃。在科技城的廢墟之上，他是絕對的「訊號主宰」，任何試圖反抗的文明火種，都將在他的電子風暴中被徹底抹除。

「肉體的迅捷終有極限，唯有將靈魂上傳至鋼鐵與光訊，方能執掌永恆的瞬滅。」
Attributes
boss屬性 hp:180, speed:15, score:45
Skills - Page 1

boss技能一【位元過載・頻段崩毀擊】 - 【極光・定頻瞬滅波】

描述： 「這是來自數據頂層的絕對執行，在預設的座標中，任何閃避都將被計算為無效代碼。當藍色的死線劃破空間，生者的存在將被強制歸零，成為這座廢墟中一段被抹除的殘留資訊。」

視覺感： 雷霆將雙翼猛然張開，在戰場上出現數道極深藍色閃電的橫向光束噴湧而出。光束中心是純粹的白熾，帶著撕裂電路的轟鳴感橫掃固定路徑，留下久久不散的焦灼氣味與電子干擾紋路。

Skills - Page 2

boss技能一【位元過載・頻段崩毀擊】 - 【亂源・數據負載爆】)

描述： 「當秩序在負載中崩潰，混亂便成了最優的武裝。這是不具備任何規律的電磁風暴，每一道流竄的電訊皆在渴求著接觸，讓汝在無序的閃光中感受數據溺斃的恐慌。」

視覺感： 雷霆仰天長，隨即，空間中出現多橫向雷電光束以隨機的高度、參差不齊的速度噴射而出。整個戰場被忽明忽暗的藍光充斥，呈現出一種電子系統崩潰邊緣的混亂美學與極致的壓迫感。

Skills - Page 3

boss技能二【高頻過載・極光積雷陣】 - 雷霆將胸口的能源核心切換至「過載模式」，雙翼高速振顫釋放高頻超聲波，引導科技城上方的冷卻介質與游離電能迅速液化，形成具備機械結構核心的「脈衝積雲」。這些雷雲本身便是高效的能量電容器，會自動鎖定區域內的移動訊號，並以「零延遲」的速度向下宣洩高達數百萬伏特的離子雷擊。這不僅是物理性的電擊，每一次雷鳴都會引發局部空間的「訊號靜默」，讓挑戰者的神經傳導與裝備反應陷入短暫的當機狀態。

描述： 「這是來自天空端的強制格式化，在不斷跳動的藍色電火中，一切物質的代碼都將被強行重寫。在此『脈衝積雲』下，雷鳴即是系統的哀鳴，展現出足以將任何防禦系統瞬間燒毀的絕對壓迫感。

視覺感： 雷霆全身裝甲縫隙噴發出幽藍色的雷光，胸口核心呈現高頻閃爍的白光。隨即，戰場上方凝聚出數朵巨大的、邊緣鑲嵌著金屬齒輪與藍色電路的「脈衝積雲」。雷雲中心燃燒著深藍色的電子火花，並以極高的頻率向地面投射出粗大的、帶有強烈像素閃爍感的淺藍色離子雷擊。

Skills - Page 4

boss技能三【離子過載・萬雷崩羽破】 - 雷霆強行將全身能源矩陣的電力灌注進機甲羽翼，使每一片金屬羽毛都處於高壓離子的包裹之下。隨著它猛然振翅揮擊，無數負載著強大電流的「離子羽刃」脫離翼尖，形成一片覆蓋戰場的致命彈幕。這些羽刃在行進中會互相感應，產生密集的電漿網，凡是被擊中的目標，其生命特徵將被強大的電磁訊號瞬間覆蓋，在絕望的電子轟鳴聲中徹底消融。

描述： 「這是來自雲端的最後判決。當萬羽齊發，汝等的存在將被重寫為崩壞的雜訊。在不熄的藍色閃光中，感受何謂絕對的資訊壓制，化為這座廢墟中一段無人知曉的殘缺代碼。」

視覺感： 雷霆於半空中發出震撼電路的低鳴，藍色機甲雙翼噴發出離子光輝，周圍空氣因高壓而閃爍著細小的電弧。隨即，成群的離子羽刃以密集的態勢向前方全域噴湧。

Skills - Page 5

boss技能四:【電磁獵殺・離子劫雷爪】 - 雷霆透過內部的戰術電腦，將高壓離子能瞬間灌注至腿部的機甲腳爪。重構成具備獨立運算核心與推進系統的鳥爪型導彈。隨著機體關節的重組與釋放，這對腳爪會脫離本體，轉化為具備「獵殺本能」的追蹤導彈。當獵殺指令下達，這些脫離本體的「掠雷爪」會自動捕捉戰場上最微弱的生物電訊號，在超頻電漿推進器的加速下，於空中劃出刁鑽的弧度進行精準打擊。凡是被掠食爪鎖定的目標，將體會到被「訊號主宰」親自抓捕、無處可逃的絕望感。

描述： 這是來自雲端的鎖定指令。當離子重構的獵手出擊，汝等的訊號早已被烙上死亡的標記。在此不熄的藍色閃光中，逃避已無意義，唯有徹底的粉碎才是唯一的終點。

視覺感： 雷霆於空中發出刺耳的電子音，雙足腳爪爆發出耀眼的藍白色像素光芒。隨著一陣金屬重組的清脆響聲，兩枚表面流淌著高壓電路紋路的掠爪雷彈脫離本體這些爪彈在行進中會根據玩家的位移不斷修正軌跡。
