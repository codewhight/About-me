Unity 橫向 2D 彈幕遊戲

使用 Unity 開發的橫向 2D 彈幕遊戲，重點在角色移動、彈道設計、 閃避節奏與關卡壓力曲線。本頁為專屬介紹，與 Python 小遊戲集為不同專案。

查看詳情 回到作品集

📌 專案說明

此 Unity 作品已從「Python 小遊戲集」中獨立出來，主題為橫向 2D 彈幕玩法，便於在履歷／作品集單獨展示遊戲迴圈設計、 子彈生成與碰撞判定。若之後有 WebGL 建置檔，可在此頁嵌入遊玩或提供下載連結。

🎯 核心玩法

玩家在橫向場景中移動並持續閃避彈幕，同時抓準時機輸出。遊戲節奏強調「走位、預判、資源分配」三者平衡，讓每一波敵彈都有可讀性與壓迫感。

🗺️ 關卡設計

  • 訓練關：引導玩家熟悉移動、閃避與射擊節奏。
  • 挑戰關：加入複合彈道與敵人組合，提升操作密度。
  • Boss 關：多階段彈幕模式，考驗讀圖與臨場反應。

⚙️ 系統機制

  • 玩家控制：橫向位移、射擊、無敵幀閃避。
  • 彈幕管理：不同發射頻率、角度與速度組合。
  • 判定與反饋：碰撞檢測、受擊提示、分數統計。

👾 敵人與彈道

  • 直線彈：高頻率基礎壓制，迫使玩家持續走位。
  • 散射彈：扇形覆蓋，限制安全區域。
  • 追蹤彈：短時間追尾，考驗節奏切換與路徑規劃。

🎒 道具與成長

  • 補給包：恢復生命值，提高容錯率。
  • 火力強化：短時間提升攻擊輸出。
  • 護盾模組：吸收一次受擊，增加通關穩定度。
