Stage 1 Boss

【不動金剛】· 磐石

磐石是伊格尼斯最強韌的盾牌，守護著軍團的最後防線。磐石的羽毛並非生物的羽組織，而是吸取了「煉獄峽谷」深處的稀有金屬與地脈靈氣凝結而成的**「熔岩金鱗」**。它在空中飛行時並不輕盈，反而帶著雷霆萬鈞的破空聲，每一次落地都能引發戈歐勒大陸的板塊震動。

它生性沉默，卻擁有無與倫比的防禦力。在侵略戰爭中，它像是一座移動的堡壘，正面接下了恐龍族勇士們所有的反擊，並將那些絕望的攻擊化作無用的火花。只要磐石屹立不倒，克羅姆平原的希望就永遠無法觸及那片被火龍統治的天空。

「凡人渴望撼動群山，卻不知群山亦有雙翼，能將所有反抗者壓成齏粉。」
Attributes
boss屬性: hp:200, speed:2, score:60
Skills - Page 1

boss技能:【萬劫碎石・千鈞碎落陣】

作為「空翼軍團」中最沈穩的守護者，磐石不需要華麗的閃電或狂暴的火焰。它只需輕輕振翅，便能引發大地的共鳴，將隱藏於雲層中的太古巨岩召喚而至。每一顆落石都承載著「戈歐勒」大陸萬年的厚重，以撕裂空氣的傾斜角度墜落。這不是攻擊，而是大地對凡人的傲慢所施予的制裁。

Skills - Page 2

boss技能:【萬劫碎石・千鈞碎落陣】 - 崩星落石・星墜沉陷

描述： 「自天之左翼傾瀉而下的毀滅之石，帶著星辰墜落的軌跡。當這股沉陷之力由左向右橫貫戰場，凡間的秩序將在塵埃中重組，唯有絕望在裂縫中蔓延。」

視覺感： 數顆佈滿裂紋、閃爍著微光的巨型落石以 45 度角由畫面左上方滑落。

Skills - Page 3

boss技能:【萬劫碎石・千鈞碎落陣】 - 地鳴落石・天傾重壓

描述： 「蒼穹在右側崩塌，太古巨岩帶著天傾之勢強勢襲來。這是不可逆轉的因果，任何企圖向上爬行的靈魂，都將在重壓之下回歸泥土的懷抱。」

視覺感： 岩塊從畫面右上方成群呼嘯而下，朝左下方形成一道道沉重的壓制線。
