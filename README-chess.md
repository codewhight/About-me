# Chess 西洋棋遊戲

這是結合 `chess.js` (負責完美的走法檢查與遊戲狀態) 與獨家設計的前端介面 (純 Vanilla JS + CSS) 所完成的西洋棋。

## 功能特色
- **完整的西洋棋規則**：
  - 支援「入堡 (Castling)」。
  - 支援「吃過路兵 (En Passant)」。
  - 支援「兵升變 (Promotion)」*(目前版本為求操作簡便，強制預設升變為「后」)*。
  - 正確的將軍 (Check) 與將死 (Checkmate)、逼和判定。
- **遊玩模式切換**：
  - **雙人遊玩 (PVP)**：共用畫面下棋。
  - **單人對抗 AI (PVE)**：AI 對手由自帶的 JavaScript Minimax 函式驅動。
- **UI 與視覺體驗**：
  - 高亮點擊的棋子與其「合法步點」。
  - 強調最後走步與被將軍的王。
  - 右側面板提供 PGN (Portable Game Notation) 實時紀錄。
- **內建 AI (Minimax)**：
  - 使用 Alpha-Beta 剪枝的 Minimax 樹尋找最佳步。
  - 使用 Piece-Square Tables 為AI提供「搶佔中心點」等基本走位觀念。

## 操作方式
1. 選擇要雙人或是對抗 AI。
2. 點擊你的棋子，它和能合法移動的方格都會亮起。
3. 把棋子落在高亮的方格上以完成移動！
4. 任何時刻都可以使用「悔棋」功能重試策略。
