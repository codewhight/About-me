# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-05-06

### Added
- **Detailed Character Skills**: Added comprehensive descriptions and names for "Normal Attack" and "Ultimate" skills for all 6 playable characters (Magmus, Flameheart, Gloomwalker, Ray-Focus, Umbra, Aelion).
- **Expanded Scene Lore**: Enhanced descriptions for all 6 game locations (Chrome Village, Training Camp, Sacred Hall, Cracked Canyon, Gloom Forest, Nova Tech City) with lore extracted from the game script.
- **Bilingual Titles**: Added English subtitles to all scene and item titles for a more professional encyclopedia feel.

### Changed
- **UI Visual Alignment**: Centered all card titles and tags for improved symmetry and visual balance across all encyclopedia categories.
- **Image Transparency**: Processed 34 relic and prop image assets to remove white backgrounds, enabling seamless integration with the dark glassmorphism theme.
- **Image Container Style**: Standardized card image wrappers in the Relics and Props sections to use a transparent background.

### Fixed
- Fixed an issue where some relic images were showing a white box against the dark theme.
- Corrected several text descriptions to strictly match the latest version of the 2D game script.

*Note: This update finalizes the Encyclopedia UI and content mapping.*
