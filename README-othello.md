# Othello 黑白棋遊戲

這是以純 JavaScript 編寫的現代黑白棋（又稱反白棋、Reversi）網頁遊戲。

## 功能特色
- **遊玩模式支援**：
  - **雙人遊玩 (PVP)**：在同一個畫面上輪流下棋。
  - **單人對抗 AI (PVE)**：玩家執黑子先手，由腳本 AI 執白子後手。
- **合法步提示系統**：
  - 玩家可在介面中開啟「提示合法步」功能，畫面將以微光亮起所有合法可下的位置。
- **AI 戰略設計**：
  - 以角（Corner）優先為策略核心（角落有加權）。
  - 自動避開邊角旁的危險位置（星位與 C 位減權）。
  - 對手行動力極小化（降低對手之後的合法步數量）。
- **自動判斷完局與 Pass 機制**：
  - 如果一方無合法步可走，系統自動跳過 (Pass) 其回合。若雙方皆無合法步，自動結算雙方的棋子數量並宣布勝負。

## 如何遊玩
1. 棋盤採用標準 8×8 網格。遊戲開始時，中心已放置交叉相間的兩黑兩白。
2. 黑子必定作為先手。
3. 只能將棋子下在能夠夾擊對方棋子的合法位置上：在橫、豎或斜方向上，用自己顏色的棋子兩端包夾住對方的棋子（期間不能有空位）。
4. 被夾住的對手棋子會全部翻轉成你的顏色。
5. 想盡辦法佔領場上的四個角落，因為四個角不會被任何人翻轉回來。
6. 全盤填滿或雙方無法下子時，場上擁有較多己方棋子者勝出。
