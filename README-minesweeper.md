# Minesweeper 踩地雷遊戲

這是一款基於 Python (Pyodide) 運行在瀏覽器中的經典踩地雷遊戲。

## 功能特色
- **多種難度選擇**：
  - 簡單（9×9，10顆地雷）
  - 普通（16×16，40顆地雷）
  - 困難（16×30，99顆地雷）
- **首點不死機制**：確保玩家第一次點擊的格子絕對不會是地雷（點擊後才隨機生成地雷）。
- **右鍵插旗**：可以標記可疑的地雷區域（以 🚩 顯示）。
- **自動展開**：當點開的格子周圍地雷數為 0 時，會自動展開附近安全的區塊。

## 如何遊玩
1. 左鍵點擊格子以揭開。若踩到 💣 則遊戲失敗。
2. 格子上的數字代表周圍 8 格內隱藏的地雷數量。
3. 利用滑鼠右鍵點擊（或長按）來為已確認的地雷位置插上旗子。
4. 揭開所有非地雷格子即可獲得勝利。

## 技術實現
- 遊戲核心邏輯以 Python 撰寫（繼承自專案內的遊戲邏輯），並在前端利用 Pyodide 直接編譯並執行。
- 使用 BFS (Breadth-First Search) 實作自動展開空白格的邏輯。
- CSS Grid 佈局搭配 JS 進行 Python 實例與頁面 DOM 的溝通。
