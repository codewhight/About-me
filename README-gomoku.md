# Gomoku 五子棋遊戲

這是以純 JavaScript 開發、具備現代化 UI 與進階比賽機制的網頁版五子棋。

## 功能特色
- **遊玩模式支援**：
  - **雙人遊玩 (PVP)**
  - **單人對抗 AI (PVE)**：AI 對手具備強大的啟發式防禦與進攻演算法。
- **進階黑棋禁手規則**：
  - 開啟禁手後，強制限制先手方（黑棋）的優勢以追求遊戲公平。
  - 偵測機制包含「**長連**（連六子以上）」、「**雙三**（同時形成兩個活三）」、「**雙四**（同時形成兩個四）」。
  - 玩家若在禁手開啟狀態下嘗試下於觸犯禁手的位子，會跳出違規警告並且不允許落子。
- **悔棋系統 (Undo)**：
  - 支援回復上一動的功能。在 AI 模式下會自動連同 AI 的回合一起回復，讓玩家可以反覆練習不同策略。
- **AI 戰略設計**：
  - 基於啟發式權重：分析場上所有的「三」、「死四」、「活四」，判斷是否應該主動攻擊還是阻擋對方的必勝連線。

## 如何遊玩
1. 板面採用 15×15 的五子棋國際標準網格。
2. 黑子先走，白子後走。兩方輪流在空白交叉點上落子。
3. 目標：讓自己顏色的棋子在橫、豎或斜線任一方向上連續排列完成「剛好五子」。
4. 若選擇開啟「黑棋禁手」，請黑棋玩家多加留意系統提示，以免因觸犯長連、雙三或雙四而無法落子。
5. 棋盤滿布卻無任何一方連五，則視為和局。
