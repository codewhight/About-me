# Tic-Tac-Toe 井字遊戲

這是一款基於純 JavaScript 與 CSS3 設計的進階井字遊戲 (Tic-Tac-Toe)，擁有流暢的動畫體驗。

## 功能特色
- **遊玩模式切換**：
  - **雙人遊玩 (PVP)**：在同一設備上輪流控制 X 與 O 落子。
  - **單人對抗 AI (PVE)**：支援玩家選擇先手 (X) 或是後手 (O)。
- **無敵的 Minimax AI**：
  - 核心對抗演算法採用了電腦科學中著名的「零總和 Minimax (極小化極大演算法)」。
  - AI 確保它絕對不會輸；最佳狀況為玩家可以成功與 AI 逼平 (平手)，否則 AI 將拿下勝利。
- **無障礙體驗**：
  - 各個可點擊區塊與系統狀態皆設計有 ARIA 標籤。
  - 支援全功能鍵盤操作 (Keyboard Navigation)。

## 操作方式
1. 從畫面右上角或設定面板選擇你想遊玩的「模式」與「符號」。
2. 點擊 / Enter 確認 3×3 網格中任意一個空白格子。
3. 若你在水平、垂直、或對角線任一方向連成三子即算獲勝。
4. 全滿時若無人連線則視為平手。

## 技術實現
- 最初由 Python Pyodide 開發，現已重構為純前端技術（放置於 `Tic-Tac-Toe/` 子目錄內，與 `tic-tac-toe.html` 整合）。
- 沒有依賴任何外部框架 (Vanilla JS, CSS)。
