Unity 2D 冒險遊戲

使用 Unity 開發的 2D 冒險遊戲。本頁為專屬介紹，與網站上的 Python 小遊戲集為不同專案。

查看詳情 回到作品集

📌 專案說明

此 Unity 作品已從「Python 小遊戲集」中獨立出來，方便在履歷／作品集單獨展示引擎、場景與遊戲設計。若之後有 WebGL 建置檔，可在此頁嵌入遊玩或提供下載連結。

📖 遊戲背景

在一個充滿魔法的世界中，玩家扮演一位年輕的冒險者，必須拯救被邪惡勢力佔領的王國。故事從主角的家鄉開始，逐步展開壯大的冒險旅程。

🗺️ 遊戲場景

  • 森林場景：茂密的森林，充滿野生動物和隱藏的寶藏。
  • 城鎮場景：繁華的城鎮，玩家可以與 NPC 互動、購買道具。

👥 遊戲角色

  • 主角：一位勇敢的年輕冒險者，具有多種技能和武器。
  • 夥伴：在旅途中遇到的盟友，提供支援和特殊能力。

👾 敵人圖鑑

  • 哥布林：小型怪物，攻擊力低但數量多。
  • 巨魔：大型敵人，防禦力強但移動緩慢。
  • 巫師：使用魔法攻擊，遠程威脅。

🎒 道具圖鑑

  • 治療藥水：恢復生命值。
  • 力量戒指：增加攻擊力。
  • 隱形斗篷：短暫隱藏身形。
