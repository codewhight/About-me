# Unity 2D 冒險遊戲（獨立作品頁）

此資料夾為 **Unity 專案在作品集網站上的獨立展示頁**，與根目錄的 **Python 小遊戲集**（`games.html`、Pyodide 等）**無關**。

## 檔案

- `index.html`：作品介紹頁（使用上層 `../styles.css`、`../script.js`）

## 之後可補充

- 將 Unity **WebGL Build** 放到例如 `unity-game/build/` 後，在 `index.html` 內嵌入 `<iframe>` 或提供下載／itch.io 連結。
- 在 Unity 專案內維護原始碼與版本；此網頁僅負責展示與說明。

## 舊連結

- 若有人書籤了舊的 `unity-2d.html`，該檔會自動導向本頁。
