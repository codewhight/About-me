# Brick Breaker 打磚塊遊戲

這是一款基於純 JavaScript 與 HTML5 `<canvas>` API 開發的經典打磚塊小遊戲。

## 功能特色
- **關卡推進機制**：支援無上限升級，每過一關磚塊的排列與難度都會發生變化。
- **物理碰撞判定**：精確計算球與邊界、球與底板、以及球與磚塊之間的碰撞反應。
- **分數與難度調整**：擊碎所有磚塊晉升下一關，且底板的寬度可能會隨擊碎磚塊數量而縮小，增加挑戰性。
- **遊戲規則介面**：內建簡易圖文對照指南，視覺化說明遊戲玩法與目的。

## 操作方式
- **開始/重新開始**：點擊畫面上方的按鈕啟動或重置遊戲狀態。
- **移動球拍**：使用鍵盤的 **左方向鍵 (←)** 與 **右方向鍵 (→)** 來操作底部藍色球拍。
- **目標**：接住回彈的綠色小球，並設法讓球破壞上方所有粉色磚塊。若小球掉出畫面底部，遊戲即宣告結束 (Game Over)。

## 技術實現
- 遊戲畫面繪製全面採用 Canvas 2D API (`ctx.arc`, `ctx.rect` 等)。
- 碰撞邏輯直接於 JavaScript 內由 `requestAnimationFrame` (或 setInterval) 進行輪詢並重繪畫面 (大約每 10 毫秒一次的刷新頻率)。
- 無任何外部依賴的輕量化獨立模組。
